Week 6 — CSS Animations

This week we are focusing entirely on CSS animations. Animations allow you to bring movement and life into your web pages by changing CSS properties over time. They are defined using @keyframes and can control properties like colour, size, position, and opacity.

We will begin with very simple examples and build up to more complex ones. For further reading and many more examples, visit the W3Schools CSS Animations page .
