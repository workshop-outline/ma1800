The Birth of the World Wide Web

The World Wide Web was invented in 1989 by Tim Berners-Lee, a British computer scientist at CERN (the European Organization for Nuclear Research). His goal was to make it easier for researchers around the world to share information across different computer systems.

In March 1989, Berners-Lee wrote his first proposal for an information management system, outlining the basic ideas of hypertext and linking documents together. The system would eventually evolve into what we know today as the World Wide Web.

By 1990, working with Belgian systems engineer Robert Cailliau, Berners-Lee had created the first web browser and editor, called WorldWideWeb, and the first web server, running on a NeXT computer at CERN.

The first website, launched in 1991, explained what the web was and how to create web pages using HTML. You can still view a copy of the original site hosted at CERN.
