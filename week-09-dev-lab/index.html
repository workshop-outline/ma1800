Week 9 — JavaScript Interactivity

We begin this week with three exercises that focus on interaction inside the browser: first by hovering to reveal an image, then animating messages with timed delays, and finally placing text inside an iPhone frame to simulate a scrolling feed. These build on your HTML and CSS skills while introducing some basic principles of interactivity.

Alongside these, we will experiment with JavaScript. Rather than a full introduction to the language, these workshops use playful examples—such as revealing content with a password or simulating a chatbox—to explore what becomes possible when code starts to manipulate a webpage dynamically.

Workshops

  1. Exercise 1 — Hover to Reveal an Image
  2. Exercise 2 — Animated Messages with Delays
  3. Exercise 3 — iPhone Simulation with Scrollable Text
  4. Exercise 4 — Password-Protected Reveal
  5. Exercise 5 — Simulated Chatbox

For more references on JavaScript basics, see MDN Web Docs: JavaScript Guide .
